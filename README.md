#### 介绍
* 一个电影信息分享的微信小程序
#### 功能介绍
##### 已实现功能
* 显示电影信息,根据地理位置展现正在上映的电影,即将上映的电影和豆瓣电影top250
* 搜索电影信息
* 获取地理位置信息（使用百度地图api把经纬度转化成城市),可以在app中更新位置信息
* 获取用户信息并显示（button open-type=“getuserInfo” + wx.getUserInfo）
* 修改个人信息并保存
* 显示系统信息
* 清除缓存
